# ProducStore


## [Live link](https://productstore-e2920.web.app/) : [Server side](https://github.com/md-abu-naim/ProducStore-Server)


### Overview :- 
 A full-stack responsive website project - Time period: 2 days. <br>
My website is related to a diffrent type products. On our website, you can find all your favorite types of Electronics, Accessories, Kitchen, Fitness, and more different types of Products,

### Features 

1. You can see all our Products on My home page. and you can search your favorite products

2. You can filter by your favorite categories such as Electronics, Accessories, Kitchen, Fitness, and you can filter by brand such as Apple, Gadget Pro, TechSound, SoundWave.

3. And you can search your specific amount rang.

##### Project Technologies: React.js, Node.js, Express.js, MongoDB, Firebase, JWT, Tailwind CSS, Tanstack query.

<br> 

## How to started this project

First, install the project server:
```bash
npm install
# or
yarn install
# or
pnpm install
# or
bun install
```

Second, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```
Open [http://localhost:5173](http://localhost:5173) with your browser to see the result.
